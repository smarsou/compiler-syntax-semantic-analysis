# Arbre Syntaxique
## Pour executer l'analyse syntaxique:
    make run target=tests/succes/<....>

### Pour seulement compiler la grammaire
    make parser

# AST
## Pour afficher l'AST d'un fichier.exp
    make astrun target=tests/succes/<...>

### Pour seulement compiler la grammaire
    make astparser
